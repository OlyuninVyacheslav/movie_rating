Хронология изменений исследовательского ноутбука по курсовому проекту
«Оценка рейтингов фильмов» (MovieLens 20M, PyTorch, регрессия).

## [2026-05-30] v0.1 — Создание структуры ноутбука

### Контекст
Ноутбук был пустым (0 байт). Цель — зафиксировать каркас (markdown + пустые
code-ячейки с комментариями-заглушками) перед написанием кода.

### Что было сделано
- Создан `movie_rating.ipynb` с **50 ячейками**: 20 markdown, 30 code.
- Определена архитектура эксперимента:
  - **Задача**: гибридная регрессия `user → movie`, целевая `rating ∈ [0.5, 5.0]`.
  - **Данные**: подвыборка ~1–2 млн строк (активные пользователи).
  - **Стек**: PyTorch 2.13 (CUDA), pandas, numpy, scikit-learn, matplotlib.

### Разделы ноутбука
| № | Раздел | Ячеек |
|---|--------|-------|
| 0 | Титул | 1 MD |
| 1 | Введение и постановка задачи | 4 MD |
| 2 | Импорт библиотек и конфигурация | 2 CODE |
| 3 | Загрузка данных | 1 MD + 2 CODE |
| 4 | Разведочный анализ (EDA) | 2 MD + 6 CODE |
| 5 | Предобработка данных | 1 MD + 4 CODE |
| 6 | Отбор значимых признаков | 1 MD + 3 CODE + 1 MD |
| 7 | Проектирование и реализация стенда | 1 MD + 4 CODE |
| 8 | Нейросетевые модели (3 шт.) | 1 MD + 3 CODE |
| 9 | Подбор гиперпараметров | 1 MD + 1 CODE |
| 10 | Обучение финальных моделей | 2 CODE |
| 11 | Оценка качества и сравнительный анализ | 1 MD + 3 CODE |
| 12 | Выбор лучшей модели и анализ результатов | 1 MD |
| 13 | Выводы | 1 MD |

### Модели (заложены как заглушки)
1. **Matrix Factorization** — dot-product эмбеддингов + смещения.
2. **NeuMF (Neural CF)** — конкатенация эмбеддингов → MLP.
3. **Hybrid** — эмбеддинги + отобранные контентные признаки → MLP.
+ **Baseline** (глобальное среднее + модель смещений) как точка отсчёта.

### Метрики (определены в разделе 1)
RMSE (основная), MAE, R².

  - конфиг: `SEED=42`, `DATA_DIR`, `MIN_USER_RATINGS=50`, `TARGET_RATINGS=2_000_000`,

    отбор активных пользователей (≥50 оценок) → случайный набор юзеров до

- **Раздел 4 (EDA, 6 ячеек)**:

  - обзор genome-признаков (распределение релевантности, топ тегов по std).
